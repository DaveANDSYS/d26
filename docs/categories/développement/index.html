Créer un dépôt GitHub

September 28, 2017
Qu’est-ce que GitHub ? GitHub est un service d’hébergement de dépôts Git dans le Cloud. Il offre une interface web pratique pour gérer, éditer et partager les fichiers d’un projet sur un dépôt Git distant. A l’origine, Github a été créé pour permettre la gestion de projets Opensource de grande ampleur. Mais au fil du temps, la plateforme a attiré des projets de tous types, qu’ils soient à l’initiative d’organisations ou d’individus.
