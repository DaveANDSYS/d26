A propos de Hugo

April 09, 2014

Hugo est un générateur de sites statiques écrit en Go(lang).

Il fait usage d’autres projets open source parmi lesquels :

  • Cobra
  • Viper
  • J Walter Weatherman
  • Cast

Pour en apprendre plus à propos de Hugo, visitez GitHub.

Installation

Le portail de Hugo:

  • Développé en Go
  • Librement inspiré de Jekyll
  • Développé principalement par spf13 dans le train pendant le trajet vers et depuis Manhattan.
  • Codé avec Vim en utilisant spf13-vim

Des questions ou des suggestions ? Vous êtes libres d’ouvrir un ticket sur GitHub ou d’interpeller spf13 sur Twitter.

Merci de votre attention !
